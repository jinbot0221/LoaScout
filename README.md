# LoaScout

로스트아크 화면 공유 기반 AI 전투력 분석기 콘셉트 웹페이지입니다. 브라우저 화면 공유 기능으로 전투 화면 또는 전투력 분석기를 연결하고, 데모 분석과 장문 리포트 미리보기를 확인할 수 있습니다.

## 실행

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:5173`을 열어 확인할 수 있습니다.

## 빌드

```bash
npm run build
```

정적 파일은 `dist/` 폴더로 복사됩니다.

## 카페24 배포

카페24 웹호스팅의 파일 매니저 또는 FTP에서 `npm run build`로 생성된 `dist/` 내부 파일을 사이트 루트(예: `www/`, `public_html/`, 또는 호스팅에서 안내하는 웹 루트)에 업로드하면 됩니다. 서버 사이드 런타임이나 외부 패키지가 필요 없는 정적 페이지입니다.

> 화면 공유는 브라우저 보안 정책상 HTTPS 환경 또는 localhost에서 동작합니다. 실제 카페24 도메인에서는 SSL을 적용한 뒤 테스트해주세요.
