# LoaScout

로스트아크 화면 공유 기반 AI 전투력 분석기 콘셉트 웹페이지입니다. 게임 화면과 전투력 분석기를 연결해 스킬 사이클, 버프 유지율, 포지셔닝을 분석하고 예상 딜 상승 가이드를 제공하는 랜딩 페이지 UI를 포함합니다.

## 실행

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:5173`을 열어 확인할 수 있습니다.

## 빌드

```bash
npm run build
```

정적 파일은 `dist/` 폴더로 복사됩니다.


## 카페24 배포

카페24 웹호스팅의 파일 매니저 또는 FTP에서 `npm run build`로 생성된 `dist/` 내부 파일을 사이트 루트(예: `www/`, `public_html/`, 또는 호스팅에서 안내하는 웹 루트)에 업로드하면 됩니다. 서버 사이드 런타임이나 외부 패키지가 필요 없는 정적 페이지입니다.
